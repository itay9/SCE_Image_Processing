Script takes 3 arguments: source of left image, source of right image and output_folder,
and creates panoramic image from left and right given images.

Package needs to be install before using: 
opencv-contrib-python==4.5.4.58

Then write in your command line:
python hw3.py source_of_left_image source_of_right_image output_folder

source_left_image : path to left image file (use this pattern: "[insert_path_here]")
source_right_image : path to rightimage file (use this pattern: "[insert_path_here]")
output_folder : path to directory which contains the new image (use this pattern: "[insert_path_here]")
